Simple configurable widgetized plugin to display the number of atteendes of an Eventbrite event.

== Description ==

This plugin will display a count of atteendes of an Eventbrite event.

For example:

* Our event will have 14 participants.
* 31 have registered.
* 22 boys and girls bought tickets already...

The plugin can be used in a widgetized sidebar or in your theme.

On the configuration panel is also a donation link, so you can chip in if you like.

== Installation ==

Installation is all straight forward.

1. Download the plugin archive
1. uncompress archive
1. Upload archive to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add widget to your sidebar
